Простой веб-интерфейс для анализа логов IoT-устройства.
1. Необходимо запустить файл в index.html в любом браузере и, либо ввести IP IoT-устройства, который передает данные через websocket в формате объекта с параметрами "event","importance","device_id","timestamp","details", либо ввести слово "TEST" для симулирования логов.
2. Есть функционал поиска, сортировки, ограничения кол-ва записей, пагинации, просмотра деталей в модальном окне.
